# 串流快速入門

代理開發套件 (Agent Development Kit, ADK) 透過串流功能，讓您能與 AI 代理進行即時、互動的體驗。這使得即時語音對話、即時工具使用以及來自代理的持續更新等功能成為可能。

本頁面提供了快速入門範例，幫助您快速上手 Python 和 Java ADK 中的串流功能。
