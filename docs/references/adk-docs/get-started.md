# 開始使用

代理開發套件 (Agent Development Kit, ADK) 旨在使開發人員能夠建構、管理、評估和部署由 AI 驅動的代理。它為創建能夠處理複雜任務和工作流程的對話式和非對話式代理提供了一個強大而靈活的環境。
